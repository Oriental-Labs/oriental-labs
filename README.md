# Oriental Labs — Website

Marketing and studio website for [Oriental Labs](https://orientalabs.dev) — a Uruguayan software & AI studio.

## Stack

- **Next.js 14** (App Router) + TypeScript
- **TailwindCSS** + Framer Motion
- **Supabase** — blog posts, projects, reviews, admin settings
- **i18n** — English and Spanish (`/en/`, `/es/`)

## Features

- Homepage with hero, services, projects, testimonials, about, and contact sections
- Blog with Markdown editor and preview
- Project case study pages
- Admin panel (`/admin`) — CRUD for posts, projects, reviews, and site settings
- Maintenance mode, global announcement banner, audit logs
- Dynamic OG images via `/api/og`
